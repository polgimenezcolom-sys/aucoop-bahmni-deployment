## Patient Flow (Pre-Digital)

```
Patient arrives → Handwritten register → Wait → Triage (paper vitals)
→ Consultation (paper notes) → Lab slip (paper) → Walk to lab
→ Wait for results (paper) → Walk back to doctor → Paper prescription
→ Walk to pharmacy → Pay cashier (paper receipt) → Discharge
```

**Average patient journey time**: 3–5 hours (mostly waiting and walking paper between departments)

## Infrastructure Challenges

- Grid electricity unreliable (frequent blackouts)
- Diesel generator available but fuel expensive
- Mobile broadband (4G) with CGNAT — no public IP
- All records paper-based
- Patient ID via handwritten cards (frequently lost)
- No clinical workstations (only 1-2 admin PCs)
